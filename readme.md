Resume Skill Analyzer

--> About the Project

This project evaluates a resume against predefined job roles and provides feedback on skill coverage. It is designed as a beginner-friendly AI project using basic NLP techniques.

-->  Key Features

Processes raw resume text
Identifies relevant keywords
Calculates a skill match score
Highlights missing competencies

-->  Setup Instructions

Install Python (3.x recommended)
Install required library:
    pip install nltk
Run the script:
    python resume_analyzer.py

-->  Inputs Required

> Resume text (manual input)
> Desired job role

--> Output Generated

> Skill match percentage
> Identified skills
> Missing skills
